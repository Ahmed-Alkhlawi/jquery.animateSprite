jQuery.animateSprite

Animating sprites, the easy way

jQuery.animateSprite is a simple plugin to control sprite animations

Download plugin Visit Github page

How it works

1.

Create an empty div and assign the sprite as a background, set the width and height so it displays the first frame

.scott {
    height: 140px;
    width: 108px;
    background-image: url(img/scottpilgrim.png);
}

2.

Call the plugin using the following options

$(".scott").animateSprite({
    columns: 8,
    totalFrames: 8,
    duration: 750,
    loop: true,
    complete: function(){
        // this will be called when
        // there is no loop and the
        // animation finishes
        alert("animation End");
    }
});

3.

Use the following methods to control the animation

$(".scott").animateSprite("stopAnimation") $(".scott").animateSprite("resumeAnimation") $(".scott").animateSprite("restartAnimation") $(".scott").animateSprite("frame", 3)
